## 📖 Overview
A metacognitive AI study companion with persistent memory that tracks learning patterns and provides personalized insights to optimize student performance.

## ❓ Why This Project?
Static learning tools often fail to adapt to a student's unique mistakes. Cogni uses **Hindsight and LLMs** to maintain a long-term memory of a student's progress, identifying recurring misconceptions and providing targeted feedback to break the cycle of repeated errors.

## ✨ Key Features
- **Persistent Memory**: Tracking learning history to provide contextualized assistance.
- **Adaptive Feedback**: LLM-driven insights that evolve as the student masters new concepts.
- **Interactive Analytics**: Dashboards for confidence trends, accuracy patterns, and study effectiveness.
- **Personalized Tutoring**: AI agents that simulate an expert tutor's metacognitive strategies.

## 🏗️ Architecture
- **Memory Engine**: Hindsight integration for long-term contextual recall.
- **Inference**: High-speed LLM processing using Groq.
- **Frontend**: Next.js App Router for a seamless, fluid student interface.
